🎓 新しい運用ステップ（ワンクッション形式）
記事を書く: blog/posts/ に Markdown ファイルを追加します。
魔法を唱える（ワンクッション！）: ターミナルで以下のコマンドを打ちます。

bash
python manage.py update

これで、わたくしが記事をスキャンして posts.json を最新状態に書き換えますわ！
世界に放つ: Git でプッシュすれば、Webサイトに反映されます。
🛠️ 変更のポイント
manage.py の新設: あなたが「よし、更新しよう！」と思ったタイミングで実行できる専用コマンドを作りました。
GitHub連携の調整: 勝手に裏で動くのではなく、あなたが manage.py で生成した結果を自分の手でプッシュする（あるいはGitHub上で手動実行する）という、意志のあるフローに変更いたしました。
